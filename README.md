우선 현 사이트를 이용하는 사용자 입장에서 편리한 구조를 생각하여<br>

전체적인 디자인은 부드럽고 깔끔한 이미지를 주기 위해 둥근 테두리를 사용하고 로고의 색상을 가져와 통일감을 주었습니다.<br>
